void gpioSetWifiMode ( u16  mode)

Set the GPIO Wi-Fi chipset mode.

Using the NDS Wi-Fi chip requires GPIO_WIFI_MODE_NTR on DSi consoles. On DSi, DWM-W024 boards require GPIO_WIFI_MODE_TWL to correctly operate.
